ABOUT
-----
Dungeon Quest RPG is a classic dungeon-crawling RPG inspired by Final Fantasy
Legend and Sword of Hope. This is a test build for gameplay and balance
feedback.

SYSTEM REQUIREMENTS
-------------------
- Windows 7 or later (64-bit)
- Command Prompt or PowerShell
- No installation required - just run the .exe!

HOW TO RUN
----------
1. Double-click rpg_game.exe
   OR
2. Open Command Prompt/PowerShell in this folder and type: rpg_game.exe

The game runs in a text-based console window.

CONTROLS
--------
Movement (Dungeon):
  W - Move North
  A - Move West
  S - Move South
  D - Move East
  M - Open camp menu (heal, equipment, save)

Menus:
  W/S       - Navigate up/down
  Enter / Z - Confirm selection
  X / Esc   - Cancel / Go back

Battle:
  Navigate with W/S
  Enter/Z to select actions and targets
  X/Esc to cancel

GETTING STARTED
---------------
1. Create your party (up to 4 members)
   - Choose from 6 job classes: Knight, Black Belt, Thief, Sage, Priest, Mage
   - Each class has unique stats and abilities

2. Visit the Town
   - Inn: Rest and recover (50 Gold)
   - Item Shop: Buy potions, ethers, antidotes, etc.
   - Equipment Shop: Purchase weapons, armor, helmets, accessories
   - Tavern: Get hints and tips

3. Enter dungeons and battle monsters
   - Complete 4 dungeons to collect crystals
   - Defeat dungeon bosses for key items
   - Final Sanctum unlocks when all 4 crystals are collected

JOB CLASSES
-----------
Knight      - High HP, strong physical attacks, defensive skills
Black Belt  - Balanced fighter, critical hit specialist, MP regeneration
Thief       - Fast, can steal items, backstab and poison attacks
Sage        - Offensive magic (Fire spells), support magic, high INT
Priest      - Healing magic, buff spells, status effect removal
Mage        - Elemental magic (Fire/Ice/Bolt), debuffs, ultimate nukes

GAMEPLAY TIPS
-------------
- Balance your party: Have at least one healer (Priest or Sage)
- Thieves can steal consumables from enemies (25% chance)
- Enemies may drop items after battle (if not already stolen)
- Equipment matters! Visit shops between dungeons
- Use the camp menu (M) to heal with items and save your progress
- Level up to learn new skills automatically

SAVE SYSTEM
-----------
- 3 save slots available
- Save from dungeon camp menu (press M while exploring)
- Save files stored as save_slot_X.sav

KNOWN ISSUES
------------
- Console window may need to be resized for optimal display
- Some antivirus software may flag the .exe (false positive - it's safe!)
