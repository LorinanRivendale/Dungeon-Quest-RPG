=== DUNGEON QUEST RPG - CONTROLS ===

EXPLORATION
-----------
W/A/S/D     Move character
Z / Space   Interact with tile (stairs, treasure, boss)
X / Escape  Leave dungeon (with confirmation)
Enter / I   Open menu

MENU NAVIGATION (Cursor-Based)
-------------------------------
W / S       Move cursor up/down
Enter / Z   Select highlighted option
X / Escape  Cancel / Go back one level

DUNGEON MENU
------------
> Camp          Access rest, healing, and save options
> Inventory     Manage items and equipment
> Party Status  View detailed character stats
> Return        Return to dungeon exploration

CAMP SUBMENU
------------
> View Party Status  See party HP/MP/status
> Use Tent          Fully restore party HP/MP
> Save Game         Save to slot 1, 2, or suspend save
> Return            Back to dungeon menu

BATTLE (Cursor-Based)
---------------------
W / S       Move cursor up/down through actions
Enter / Z   Select action
X / Escape  Cancel selection (return to action menu)

Actions:
> Attack        Choose target with cursor
> Skill/Magic   Choose skill, then target with cursor
> Item          Choose item, then target with cursor
> Defend        Reduce damage taken this turn
> Flee          Attempt to escape (not available vs bosses)

MAP SYMBOLS
-----------
@   You
#   Wall
.   Floor
>   Stairs down
<   Stairs up
$   Treasure
B   Boss
E   Entrance

TIPS
----
- Mix physical and magic classes
- Save MP for bosses
- Use tents to rest between floors
- Characters learn skills as they level
- Explore thoroughly for treasure
